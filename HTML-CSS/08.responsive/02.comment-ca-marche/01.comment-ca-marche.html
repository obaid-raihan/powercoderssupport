CSS - Media Queries

Une media query (ou requête média) consiste en un type de média, et au moins une expression limitant la portée des déclarations CSS, en mettant à profit les particularités des supports multimédias comme leur largeur, leur hauteur ou leur affichage des couleurs. Ajouté dans CSS3, les media queries permettent d'adapter la présentation du contenu à une large gamme d'appareils sans changer le contenu lui-même.

On peut changer les media queries en fonction de...

  • Type de media (screen, print, ...)
  • Largeur/Hauteur device
  • Orientation device

Exemple - Changez la largeur de la fenêtre

J'aurai un background different à chaque point de rupture

					#box {
						width: 100%;
						padding: 3em 10%;
						text-align: center;
						background: #ccc;
					}

					/* Point de rupture 1 - tablettes paysage  */
					@media only screen and (max-width: 1024px) {
				    #box {
				      background: #AED6F1;
				    }
					}

					/* Point de rupture 2 - tablettes portrait  */
					@media only screen and (max-width: 768px) {
				    #box {
				      background: #A569BD;
				    }
					}

					/* Point de rupture 3 - phones paysage  */
					@media only screen and (max-width: 667px) {
				    #box {
				      background: #CB4335;
				    }
					}

					/* Point de rupture 4 - phones portrait  */
					@media only screen and (max-width: 480px) {
				    #box {
				      background: #2E4053;
				      color: #fff;
				    }
					}
