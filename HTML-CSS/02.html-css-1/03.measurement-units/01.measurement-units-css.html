Unités de mesure pour longeurs en CSS

Unités relatives

em

Représente la taille de police héritée de l'élément parent le plus proche
Les propriétés CSS line-height, font-size, margin-bottom et margin-top sont souvent des dimensions exprimées en em


		html {
			font-size: 20px;
		}

		html p {
			width: 20em;		=> 400px;
			font-size: 1em;		=> 20px
			line-height: 1.5em;	=> 30px

			margin-top: 1em;	=> 20px;
			padding: 2em;		=> 40px autour du texte;
		}

rem (root em)

représente la font-size de l'élément racine (la taille de la police de l'élément html (root))


		html {
			font-size: 20px;
		}

		div {
			font-size: 12px;
		}

		div p {
			font-size: 1rem;	=> 20px

Unités proportionnelles

%

Pour déterminer des tailles en fonction de la taille des éléments parent 


		div {
			width: 800px;
		}
		div p {
			width: 50%;		=> 400px
		}

Unités absolues

px

Pour l'affichage sur écran, correspond typiquement à un pixel de l'affichage
Pour les écrans en haute résolution et les imprimantes, un pixel CSS correspond à plusieurs pixels du périphérique, de sorte que le nombre de pixels par pouce (ppi) reste aux alentours de 96 (!) 


		div 1 {
			width: 200px;
			height: 50px;
		}
		div 2 {
			width: 800px;
			height: 50px;
		}
