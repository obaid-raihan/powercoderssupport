CSS - transition

Permet de définir la transition entre deux états d'un élément. Les différents états peuvent être définis à l'aide de pseudo-classes telles que :hover ou :active ou être définis dynamiquement avec JavaScript.

transition: [transition-property] [transition-duration] [transition-timing-function] [transition-delay];

Exemples

#transition-1 #transition-2 #transition-3 #transition-4 #transition-5 #transition-6 

					#transition-1 {
						border-radius: 0;
						transition: border-radius .2s linear;
					}
					#transition-1:hover {
						border-radius: 50%;
					}

					#transition-2 {
						background: #FADBD8;
						color: #333;
						transition: background .4s ease, color .4s ease-out;
					}
					#transition-2:hover {
						background: #B03A2E;
						color: #fff;
					}

					#transition-3 {
						background: transparent;
						color: #888;
						border-top: solid 3px transparent;
						border-bottom: solid 3px transparent;

						transition: border .5s ease-in-out, color .5s ease;
					}
					#transition-3:hover {
						border-top: solid 3px #333;
						border-bottom: solid 3px #333;
						color: #000;
					}

					#transition-4 {
						transition: width .2s ease-out;
					}
					#transition-4:hover {
						width: 140px;
					}

					#transition-5 {
						transition: transform .4s ease;
					}
					#transition-5:hover {
						transform: scale(1.2) rotate(45deg);
					}

					#transition-6 {
						transition: all .5s ease; 
					}
					#transition-6:hover {
						background: #8E44AD;
						box-shadow: 0 0 15px 3px rgba(0,0,0,.3);
						transform: rotate(45deg);
						border-radius: 20%;
					}

Compatibilité avec les vieux navigateurs


					#transition-1 {
						-webkit-transition: border-radius .2s linear;
						-moz-transition: border-radius .2s linear;
						transition: border-radius .2s linear;
					}
