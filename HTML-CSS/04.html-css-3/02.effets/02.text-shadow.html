CSS - text-shadow

La propriété text-shadow permet d'ajouter une ombre au texte boîte avec une liste des valuers séparées par des virgules:

offset-x | offset-y | blur-radius | color

Exemples

C’est bien facile à dire « Buvez-moi, » mais Alice était trop fine pour obéir à l’aveuglette.
