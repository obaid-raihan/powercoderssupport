CSS - box-shadow

La propriété box-shadow décrit les ombres portées appliquées à une boîte avec une liste des valuers séparées par des virgules:

offset-x | offset-y | blur-radius | spread-radius | color

Exemples

#shadow-1
#shadow-2
#shadow-3

					#shadow-1 {
						box-shadow: 0 0 20px 3px #ccc;
					}

					#shadow-2 {
						box-shadow: 5px 5px 20px 3px rgba(150, 150, 150, .5);
					}

					#shadow-3 {
						box-shadow: 1px 1px 10px 1px rgba(150, 150, 150, .5) inset;
					}

Compatibilité avec les vieux navigateurs


					#shadow-1 {
						-webkit-box-shadow: 0 0 20px 3px #ccc;
						-moz-box-shadow: 0 0 20px 3px #ccc;
						box-shadow: 0 0 20px 3px #ccc;
					}
