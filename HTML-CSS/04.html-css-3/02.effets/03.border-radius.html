CSS - border-radius

La propriété border-radius permet de définir des coins arrondis pour la bordure d'un élément. La courbure de chaque coin est définie avec un ou deux rayons de courbures qui permettent de définir un arc de cercle ou un arc d'ellipse.

border-top-left-radius | border-top-right-radius | border-bottom-right-radius | border-bottom-left-radius

Exemples

#radius-1
#radius-2
#radius-3
#radius-4
#radius-5
#radius-6

					#radius-1 {
						border-radius: 20px;
					}

					#radius-2 {
						border-radius: 20% 0;
					}

					#radius-3 {
						border-radius: 50%;
					}

					#radius-4 {
						background: transparent;
						color: #333;
						border: solid 5px #333;
						border-radius: 50%;
					}

					#radius-5 {
						border-radius: 5% 10px 20px 5px;
					}

					#radius-6 {
						border-radius: 40px 20% / 40px 20px;
					}

Compatibilité avec les vieux navigateurs


					#radius-1 {
						-webkit-border-radius: 20px;
						-moz-border-radius: 20px;
						border-radius: 20px;
					}
