CSS - z-index

La propriété z-index définit le « z-order » d'un élément positionné en absolu par rapport à d'autres éléments.

Lorsque des éléments se chevauchent, le z-order détermine l'ordre des différentes couches que formeront les éléments.

Un élément couvrira un autre élément si sa valeur de z-index est supérieure à celle du deuxième élément.

On peut faire une analogie avec les calques en Photoshop.

Exemple

Ci-dessous un exemple de positionnement avec z-index.

Les div #un et #trois ont un z-index 2

Les div #deux et #quatre ont un z-index 1


					#container .box {
						position: absolute;
					}

					#un {
						top: 60px;
						left: 30px;
						z-index: 2;
					}

					#deux {
						top: 30px;
						left: 100px;
						z-index: 1;
					}

					#trois {
						top: 60px;
						left: 170px;
						z-index: 2;
					}

					#quatre {
						top: 30px;
						left: 240px;
						z-index: 1;
					}
