CSS - La propriété display

La propriété display définit le type de boîte utilisée pour le rendu d'un élément.

display: block

Cet élément génère une boîte bloc
  • PAR DEFAUT...
  • J'ai une largeur "100%"
  • J'ai une hauteur "auto" (hauteur du text)
  • Je suis invisible sans contenu
  • balises communes: article, aside, div, figure, footer, h1(etc), header, hr, main, nav, p, ul, section, table (voir tout)
Je suis un texte dans une boîte bloc

display: inline

Cet élément génère une ou plusieurs boîtes en ligne.
  • PAR DEFAUT...
  • J'ai une largeur "auto" (largeur du text)
  • J'ai une hauteur "auto" (hauteur du text)
  • Je suis invisible sans contenu
  • balises communes: a, b, cite, code, em, img, span, strong (voir tout)

Je suis un texte dans un paragraphe. Moi aussi, mais dans une balise span. Souvent utilisé pour mettre un mot en évidence, un lien, une image comme celle-ci...

display: inline-block

Une boîte pour un élément en bloc qui « s'écoule » dans le contenu environnant comme s'il était une boîte en ligne. Un hybride entre un élément bloc et inline...

En ligne aussi mais je peut avoir un "padding" un "margin", plutôt cool.

display: flex

L'élément se comporte comme un élément de bloc et dispose son contenu selon le modèle de boîte flexible.
