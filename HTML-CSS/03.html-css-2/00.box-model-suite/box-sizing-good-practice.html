box-sizing, meilleures pratiques

Code à mettre tout en haut dans vos styles


				html {
					box-sizing: border-box;
				}

				*, *:before, *:after {
					box-sizing: inherit;
				}

Code à mettre sur une élément html ou le box-sizing est different


				element-html-cible, element-html-cible * {
					box-sizing: content-box;
				}
